실시간 실물 인증 서비스 SABER
동국대학교 2025년 1학기 공개SW프로젝트 2조

: 중고 거래 시 활용할 수 있는 실시간 거래 인증 서비스


프로젝트 목표
본 프로젝트는 중고거래에서 발생하는 사기 문제를 사전에 예방할 수 있는 실시간 실물 인증 시스템을 구축하는 것을 목표로 한다. 특히 거래 성립 이전, 구매자가 판매자의 실물 보유 여부와 제품 상태를 직접 확인할 수 있도록 하여, 정보 비대칭으로 인한 피해를 줄이고 거래의 신뢰성을 높이는 것이 핵심이다.

웹 앱 접속 링크: https://saber.p-e.kr/


🔍 기능 상세 설계
1. 🧩 실시간 인증 요청 생성 (일회성 토큰 기반)
구매자가 인증 요청을 생성하면 백엔드에서 토큰이 자동 생성됩니다.
해당 토큰이 포함된 인증 링크는 1회 접속만 유효하며, 사용자는 직접 판매자에게 링크를 전달합니다.
최초 접속자는 JWT 토큰이 브라우저 쿠키에 저장되고, 이후 네트워크 장애 등으로 페이지 이탈 시에도
제한시간 이내
유효한 토큰 존재
접속자 식별 키 일치 조건을 모두 만족하면 인증을 이어서 진행할 수 있습니다.
조건 중 하나라도 불일치하면 인증 페이지 접근이 차단됩니다.
2. 📦 품목별 인증 항목 추천 및 사용자 정의
거래 품목(전자기기, 의류, 잡화 등)에 따라 인증 항목 추천 리스트를 제공받을 수 있습니다.
사용자는 추천 항목 외에도 자유롭게 인증 항목을 추가하여 요청서를 작성할 수 있습니다.
판매자는 해당 항목을 기준으로 사진, 영상, 텍스트 코멘트를 등록하며 인증을 수행합니다.
이를 통해 구매자는 실제로 원하는 정보에 맞는 인증 결과를 받을 수 있습니다.
3. ⏰ 타이머 기반 인증 유효 시간 제한
인증 요청 시, 구매자가 인증 유효 시간(예: 5분)을 직접 설정할 수 있습니다.
판매자가 링크에 접속하면 타이머가 시작되며, 기한 내에 인증을 완료하지 않으면 실패 처리됩니다.
실패된 요청은 구매자가 재요청 버튼을 통해 재진행할 수 있습니다.
이를 통해 사전 촬영, 미리 준비된 이미지 사용 등을 방지할 수 있습니다.
4. 🖼 인증 미디어 워터마크 및 캡처 방지
판매자가 촬영한 사진에는 자동으로 촬영 시각 정보가 삽입되어 S3에 업로드됩니다.
인증 결과를 조회할 때는
우클릭 방지
무작위 인증 코드 워터마크 삽입 기능이 적용되어 이미지 유출과 2차 사기 행위를 차단합니다.
5. 📊 인증 상태 실시간 모니터링
인증 요청의 상태는 실시간으로 갱신됩니다:
요청됨, 진행 중, 완료, 실패, 만료
상태 정보는 비동기 API 호출을 통해 서버와 주기적으로 동기화됩니다.
사용자에게 직관적인 상태 표시를 제공하며, 실패 시 재요청 버튼으로 인증을 다시 시작할 수 있습니다.
