# SABER — 실시간 실물 인증 중고거래 서비스
**동국대학교 2025년 1학기 공개SW프로젝트 2조**  
: 중고 거래 시 활용할 수 있는 실시간 거래 인증 서비스

---

## 📌 프로젝트 배경

온라인 중고거래 시장은 2021년 기준 24조 원 규모로 성장했지만, 허위 매물·이미지 도용 등 사기 피해는 연간 10만 건 이상 발생하고 있다. 기존 플랫폼의 안전결제는 거래 이후 사후 대응에 집중되어 거래 전 사전 검증이 불가능하다는 한계가 있었다. SABER는 구매자가 직접 인증 항목을 설정하고 판매자에게 실시간 인증을 요청할 수 있는 시스템을 구축하여 이 문제를 해결하고자 했다.

---

## 🛠 기술 스택

| 분류 | 기술 |
|------|------|
| Frontend | React, Nginx |

| Database | MySQL, AWS RDS |
| Storage | AWS S3 |
| Infra | AWS EC2, HTTPS |
| Design | Figma |

---

## 🔍 핵심 기능

- **일회성 토큰 기반 인증 링크**: UUID 기반 고유 링크, 최초 접속자에게만 유효
- **품목별 인증 항목 설정**: 거래 품목에 따른 추천 항목 제공 및 자유 추가
- **타이머 기반 시간 제한**: 사전 촬영·준비된 이미지 사용 방지
- **워터마크 및 캡처 방지**: 촬영 시각 삽입, 우클릭 방지, 무작위 코드 워터마크
- **실시간 상태 모니터링**: 요청됨·진행 중·완료·실패·만료 상태 실시간 갱신
